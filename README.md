# 🛒 ASHU Store - Amazon Inspired E-Commerce Website

A modern and fully responsive **Amazon-inspired e-commerce landing page** built using **HTML5, CSS3, and JavaScript**. This project focuses on creating a clean UI, smooth user experience, responsive layouts, and attractive animations while following modern front-end development practices.

## 🌐 Live Demo

**🔗 Live Website:** https://amezon-clone-rose.vercel.app/

---

## ✨ Features

- Modern Amazon-inspired UI
- Fully Responsive Design
- Sticky Navigation Bar
- Hero Banner Slider
- Search Bar UI
- Product Categories
- Featured Products
- Trending Products
- Best Selling Products
- Special Deals Section
- Brand Showcase
- Customer Reviews
- Newsletter Subscription
- App Promotion Section
- Premium Footer
- Smooth Animations
- Scroll To Top Button
